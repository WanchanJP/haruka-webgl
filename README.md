# haruka-webgl

WebGLベースの3Dプロジェクトです。

## 概要

haruka-webglは、WebGL技術を使用した3D描画アプリケーションです。現在は初期セットアップ段階にあります。

## 開発セットアップ

### 前提条件

- Node.js (推奨バージョン: 16以上)
- npm または yarn

### セットアップ手順

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## アーキテクチャ

### 想定される構成要素

- **レンダリングパイプライン**: WebGLコンテキストセットアップ、シェーダーコンパイル、描画ループ
- **シーン管理**: 3Dオブジェクト階層、カメラ制御、シーングラフ  
- **シェーダーシステム**: 各種描画エフェクトのための頂点・フラグメントシェーダー
- **アセットパイプライン**: テクスチャ、モデル、音声ファイルの読み込みとリソース管理
- **入力ハンドリング**: マウス、キーボード、ゲームパッド対応の3Dインタラクション
- **パフォーマンスシステム**: フレームレート最適化、LODシステム、メモリ管理
